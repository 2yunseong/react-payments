# 보안코드 입력(AddCardSecurityCodeInput)

카드 보안코드(CVC/CVV)를 입력하는 입력 창입니다.

보안코드가 무엇인지 익숙하지 않을 사용자를 위해서 우측에 툴팁을 제공합니다.
