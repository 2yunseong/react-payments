# 로딩 스피너(LoadingSpinner)

로딩스피너는 카드 등록 및 카드 별명 등록 시 화면에 보여지는 Loading Indicator 입니다.

사용자가 등록한 카드회사의 브랜드 칼러로 커스텀해 보여집니다.
